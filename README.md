# Notes App

A clean, minimalist Android Notes application built to demonstrate modern Android development practices.

## 🚀 Features
* Create, Edit, and Delete notes.
* Persistent storage using **Room Database**.
* Reactive UI updates using **LiveData**.

## 🛠 Tech Stack
* **Language:** Java
* **Architecture:** MVVM (Model-View-ViewModel)
* **Database:** Room
* **UI Components:** RecyclerView, CardView, Material Design 3
